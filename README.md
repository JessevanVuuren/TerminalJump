# TerminalJump
A simple terminal game made in C++


make your own level in the provided game.txt file and run the program.
every level is made out of 30 rows and 100 columns. You can use special characters for animations and effects.

animations and effects:
- %  - portal to the next level
- /\ - fire
- \# - platform
- ~ - water
